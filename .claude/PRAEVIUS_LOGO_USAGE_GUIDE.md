# Praevius Logo Usage Guide

## Complete Logo Package

Your Praevius brand identity includes 7 SVG logo files optimized for different use cases.

---

## Logo Files Overview

### 1. **praevius-logo-full.svg** (280×64px)
**Primary Logo - Charcoal Text on Light Backgrounds**

**Use for:**
- Website headers (light mode)
- Email signatures
- Documents and PDFs
- Print materials on white paper
- Light-colored backgrounds

**Colors:**
- Chevron: Orange (#FF9900)
- Wordmark "Praevius": Charcoal (#2C2C2C)
- Extension ".app": Charcoal at 65% opacity

**Minimum Size:** 120px width

## Logo Placement Rules

### Minimum Sizes

| Format | Minimum Width | Minimum Height |
|--------|--------------|----------------|
| Full Logo | 120px | 32px |
| Compact Logo | 100px | 28px |
| Icon Only | 24px | 24px |
| App Icon | 48px | 48px |

### Color Variations

- **Light backgrounds:** `praevius-logo-full.svg` (charcoal text)
- **Dark backgrounds:** `praevius-logo-white.svg` (white text)
- **Navigation:** `praevius-logo-compact.svg`
- **Favicon:** `praevius-icon-orange.svg`

---

## Don'ts - Logo Misuse

1. ❌ Don't change colors (keep orange #FF9900)
2. ❌ Don't distort or stretch
3. ❌ Don't add effects (shadows, glows)
4. ❌ Don't modify the chevron design
5. ❌ Don't scale below minimum sizes

---

## CMYK for Print

- **Orange:** C=0, M=40, Y=100, K=0 (Pantone 144 C)
- **Charcoal:** C=0, M=0, Y=0, K=82 (Pantone 426 C)
