## Technical Specifications

### Recommended Sizes

| Image | Dimensions | Format |
|-------|-----------|--------|
| Hero Dashboard | 1200 x 800 px | PNG/WebP |
| Mobile Mockup | 400 x 800 px | PNG |
| Report Preview | 800 x 1000 px | PNG |
| Background | 1920 x 1080 px | JPG/WebP |
| OG Image | 1200 x 630 px | PNG |
| Integration Visual | 800 x 600 px | PNG/SVG |

### Style Guidelines

- **DO:** Use brand colors (especially orange #FF9900)
- **DO:** Keep interfaces clean and modern
- **DO:** Show realistic but idealized UI
- **DO:** Use subtle shadows and depth
- **DON'T:** Include readable fake data/text
- **DON'T:** Use competing brand logos directly
- **DON'T:** Make UI look cluttered or dated
- **DON'T:** Use stock photo clich√©s (handshakes, etc.)
